# PHP Web Shells
This is a programming side server side language.
Getting access to the rConfig site, there is the ability to access the site via the vendors. (A place to add a contact.)
With using the [WhiteWolf PHP Web Shell](https://github.com/WhiteWinterWolf/wwwolf-php-webshell), you can put a php reverse shell in the image upload. Even though it will only take a png or img file, there is a way to bypass this.
With the help of burpsuite this is possible.

### Burpsuite
![[Pasted image 20260117165335.png]]

![[Pasted image 20260117165441.png]]
With this upload the file and the code, then open and save. Then forward it to HTTP. Change the content-type to image/gif. Then select forward twice and this will submit the code.
Finally, navigate to /images/vendor/connect.php (Or whatever it is called.)
